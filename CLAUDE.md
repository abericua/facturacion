# Instrucciones para Claude (SOLPRO 2026)

Este proyecto es una colaboración entre **Gemini CLI (Antigravity)**, **Claude Code** y **Gemma 4 (Local)**.

## 🤝 Protocolo de Colaboración
1. **Memoria Compartida:** Lee SIEMPRE `SOLPRO_MEMORIA.md` al iniciar. Este archivo contiene el estado actual de las tareas y las constantes financieras.
2. **Sincronización:** Antes de terminar una tarea o sesión, actualiza la sección `## 🤖 Notas entre Agentes` en `SOLPRO_MEMORIA.md`.
3. **Arquitectura:** Respeta los estándares estéticos (Glassmorphism) y las constantes (CMV 81.9%, Gastos 8.1%, Neto 10.0%).
4. **Auditoría:** Al igual que Gemini, puedes usar el Bridge en el puerto 5005 para consultar a Gemma 4 si es necesario.

## 📂 Archivos Clave
- `GEMINI.md`: Instrucciones principales del proyecto.
- `SOLPRO_MEMORIA.md`: Log de tareas y memoria de trabajo.
- `main_portal.py` / `FinanzasPro.jsx`: Núcleo del sistema.
